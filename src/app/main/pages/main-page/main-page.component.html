OpenLayers Angular

high-performance, feature-packed library for all your mapping needs.

OVERVIEW

OpenLayers es una libreria JavaScript OpenSource que ha sido desarrollada para el uso de información geográfica de todo tipo, publicada bajo la licencia BSD de 2 cláusulas (también conocida como FreeBSD), su objetivo es facilitar el uso de mapas dinámicos en paginas web.

Esta aplicación de Angular ha sido construida completamente de forma modular utilizando LazyLoad, en ella se implementa un Módulo específico para integrar OpenLayers de manera fácil a proyectos de Angular utilizando componentes.
